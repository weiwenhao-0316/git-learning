\# MD QA Assistant Day 01



今天做了什么：



\- 创建了一个本地 Markdown 问答助手项目骨架

\- 创建了 data 和 src 目录

\- 用 Python 读取本地 notes.md 文件

\- 在命令行输入问题

\- 把问题和资料内容打印出来



我学到了什么：



\- AI 问答助手的第一步不是先上模型，而是先把输入、资料、上下文组织起来

\- `Path()` 可以帮助处理文件路径

\- `read\_text()` 可以读取文件内容

\- 一个 AI 项目也可以先从很小的骨架开始



我现在的理解：



\- 问答助手本质上是“问题 + 资料 -> 给模型 -> 返回答案”

\- 我今天已经完成了这条链路的前半段

\- 后面只差把这些内容发给模型 API



下次要做：



\- 接入模型 API

\- 让模型根据 notes.md 回答问题
