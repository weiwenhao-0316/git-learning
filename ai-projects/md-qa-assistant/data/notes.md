\# Python Learning Notes



\## input

input() is used to get user input.

It lets the program wait for the user to type something.



\## print

print() is used to display output.

It shows text or variable values on the screen.



\## if

if / elif / else can be used for conditional logic.

They allow the program to make decisions.



\## for

for can be used for loops.

It is useful when you want to repeat an action multiple times.



\## variable

A variable is used to store data.

For example, name = "Tom" stores a string in the variable name.
